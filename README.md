# Chill-O-Meter

Offline Desktop App fuer Stress-Checks, gefuehrte Uebungen und Fokus-Sessions.

## Features

- Stress-Check mit Verlauf
- Gefuehrte Uebungen:
  - Physiologischer Seufzer
  - Box-Breathing
  - 5-4-3-2-1 Grounding
  - Progressive Muskelentspannung
- Fokus-Modus mit Timer
- Dashboard mit Statistiken
- Lokale Datenspeicherung (keine Cloud)


## Datenschutz

Die App speichert Nutzungsdaten lokal auf dem Geraet (z. B. Einstellungen, Stressverlauf, Uebungsverlauf) nur zum Zweck der App-Funktion.
Beim ersten Start wird eine Zustimmung abgefragt.
